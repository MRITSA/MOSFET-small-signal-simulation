# MOSFET-small-signal-simulation
This projects simulates the small signal behavior of mosfet using python. It analyzes Gm variation and frequency response characteristics 

#Technologies Used
- Python
- NumPy
- Matplotlib

 Theory Used

Frequency Response:
A(f) = Av / sqrt(1 + (f/fc)^2)
Transconductance:
gm = 2Id / (Vgs - Vth)
